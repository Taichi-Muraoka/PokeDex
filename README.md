## 準備

#### git clone
任意のディレクトリで以下コマンド実行
```
git clone git@github.com:Taichi-Muraoka/PokeDex.git
```

#### npm install
クローンしてきたプロジェクトフォルダ内で以下コマンド実行
```
npm install
```

#### npm run start
同じくクローンしてきたプロジェクトフォルダ内で以下実行
```
npm run start
```
